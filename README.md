# DarkByte Labs — Google Play Organization Website

Ultra-modern Play Store organization portal for DarkByte Labs.

## 🚀 Deploy to GitHub Pages

1. Create a repo (e.g. `darkbytelabs.github.io` or any repo name)
2. Push all files in this folder to the `main` branch
3. Go to **Settings → Pages → Source → Deploy from branch → main / root**
4. Your site will be live at `https://<username>.github.io/<repo>/`

| File | Purpose |

## 🎨 Design

- **60% Royal Purple gradient** — primary brand colour
- **40% Deep Blue** — supporting accent
- **10% White** — text and highlights
- Glassmorphism navbar, ambient orb background, scroll-reveal animations
- Google Fonts: Outfit + Space Grotesk

## 📋 Sections

- Hero with Play Store badge + floating logo
- Organization stats bar
- About / values
- Published apps portfolio
- Google Play compliance checklist
- Privacy Policy (Play Store ready)
- Terms of Service
- Contact section
